# :package: 未定研 Shell Arc 2026 コマンド一覧
---

## :wrench: 1. Shell Arc

**`..up`**
> メッセージにファイルを添付して送信すると、提出確認UIが表示されます。
> UIに従って選択すると、ファイルが提出されます。

**`..upbig`**
> Discordで送信できない大容量ファイルを提出したいときに使用します。
> アップロード用HTMLページが生成されます。それのファイルをダウンロードして、開いてアップロードしてください**:warning: 300秒以内にアップロードしてください。**

**`..appr`** 〔作画監督等向け〕
> 送信すると、そのチャンネルのカットを承認するか否かを選択するUIが表示されます。
> UIに従って選択することでチェックができます。

**`..dl (バージョン)`**
> デフォルトで該当カットの **「最新確定カット」** を返します。
> - `-1` を指定 → **「最新チェック待ちカット」** を返します
> - バージョンインデックスを指定 → **指定バージョン** を返します

**`..check`** 〔作画監督等向け〕
> デフォルトで **「最新チェック待ちカット」** を返します。
> 作監チェックなどに便利です。

**`..reg (-f)`**
> デフォルトで自分を該当カットに登録します。
> `-f` フラグを付けると、既に登録者がいる場合でも **強制的に自分の名前で上書き** できます。

**`..history 作業工程 (最大長さ)`**
> 指定した作業工程の提出履歴を確認できます。
> - 最大長さを指定すると、返される件数を制限できます
> - 返される形式：`バージョンインデックス - 履歴内容`
> - このインデックスを使い、`..dl` で過去バージョンを取得できます

**`..ask (メンバー)`**
> デフォルトで **自分の現在の担当作業** を確認できます。
> メンバーを指定すると、**そのメンバーの担当作業** を確認できます。

**`..sync`**
> サーバー内容をクラウドに同期します。

**`..sapyc コマンド`**
> ShellArc Expression の命令系コマンドを実行できます。

## :speech_balloon: 2. ぬるたろうchat

**`..nuru 内容`**
> テクニカルサポートチャットボット **「ぬるたろう」** を呼び出します。
> ShellArc の使い方など、技術的な質問に対応しています。

## :date: 3. Itemi Action

**`..remind YYYYMMDDHHMM リマインド内容 (メンバー番号)`**
> いてみちゃんにリマインダーを登録できます。
> 指定した日時に、登録したチャンネルへリマインドが届きます。
> デフォルトのリマインド対象は **送信者自身** です。

**`..itemi 内容`**
> 制作進行サポートチャットボット **「いてみちゃん」** を呼び出します。
> スケジュールについて相談できます。
